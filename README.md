## Installazione

1. Clona il repository:

```bash
git clone https://github.com/tuousername/fitpdf.git
cd fitpdf
```

2. Installa le dipendenze:

```bash
npm install
# oppure
yarn install
```

3. Avvia il server di sviluppo:

```bash
npm run dev
# oppure
yarn dev
```

4. Apri [http://localhost:3000](http://localhost:3000) nel tuo browser.
